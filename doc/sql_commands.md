# SQL Commands

Since commands are generated by the table definitions and queries, I had to copy them from the logs. The commands are formatted with newlines and tabs so that they are easier to read.

## CREATE

**Campus**:
```sql
CREATE TABLE IF NOT EXISTS campus
(
  name TEXT NOT NULL UNIQUE,
  address TEXT,
  key INTEGER PRIMARY KEY AUTOINCREMENT
);
```

**Building**:
```sql
CREATE TABLE IF NOT EXISTS building
(
  name TEXT NOT NULL,
  building_type TEXT,
  location TEXT,
  campus_key INTEGER,
  key INTEGER PRIMARY KEY AUTOINCREMENT,
  FOREIGN KEY (campus_key) REFERENCES campus(key) ON DELETE CASCADE
);
```

**Allocation**:
```sql
CREATE TABLE IF NOT EXISTS allocation
(
  room_key INTEGER,
  event_type TEXT,
  event_key INTEGER,
  start_time TIMESTAMP NOT NULL,
  end_time TIMESTAMP NOT NULL,
  key INTEGER PRIMARY KEY AUTOINCREMENT,
  FOREIGN KEY (room_key) REFERENCES room(key) ON DELETE CASCADE,
  CHECK (start_time < end_time)
);
```
